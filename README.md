Surveyor NeoForge Mod
=======

Minimal NeoForge mod scaffold for The Arcane Frontier modpack instance.

Targets:
- Minecraft 1.21.1
- NeoForge 21.1.222
- Java 21

Common commands:
- `gradlew build`
- `gradlew runClient`
- `gradlew runServer`

Output jars are written to `build/libs`.

Additional resources:
- NeoForge docs: https://docs.neoforged.net/
- NeoForged Discord: https://discord.neoforged.net/
